# PSPLIB data

`sample_j6.sm` is a small synthetic instance in PSPLIB `.sm` format, included
so the parser and tests run without any download.

To use the real benchmarks, download the single-mode sets (j30.sm, j60.sm,
j120.sm) and their optimal/best-known bound files from the PSPLIB library and
place the `.sm` files in this folder. The parser handles them unchanged.
